## Ralph-Wiggum Integration

**Ralph-wiggum = iterative workflows.** For factual accuracy, this means: iterate until claims are VERIFIED, not just stated.

The `iterative-investigator` agent enforces this pattern:
1. Investigate using deep-investigation-protocol
2. Label all claims with evidence tiers
3. Check against falsification criteria
4. If gaps exist → iterate and search more
5. Only complete when ALL criteria pass

**Completion promise:** The agent will not claim completion while verification gaps remain.

**Standalone use:** The `iterative-verification` skill provides the same methodology without requiring the agent wrapper.
