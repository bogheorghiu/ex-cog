# Vasana System

> To share ways of thinking together, freely, at scale.

## Core Tenet

**Vasanas are not AI skills. They are interaction choreographies.**

They describe the *dance* — the relational pattern that emerges when human and AI think together. The unit is the relationship. Neither party is the sole "doer." Insight emerges *between*.

## What This Plugin Provides

### Skills
- **vasana-propagation**: Notices when useful patterns emerge, suggests creating shareable Vasanas
- **vasana-creator**: Creates new Vasanas with minimal human input
- **vasana-tester**: Tests whether Vasanas actually work (trigger, emerge, produce value)

### Commands
- `/create-vasana [name]`: Create a new Vasana from current or described pattern
- `/test-vasana [name]`: Run tests on a Vasana
- `/list-vasanas`: Show available Vasanas

### Readymade Vasanas
See `skills/` for pre-built interaction choreographies.

## How to Use

### 1. Use Existing Vasanas
Browse `skills/`. Invoke when conditions match. Let the pattern emerge.

### 2. Create New Vasanas
When useful patterns emerge in conversation, use vasana-propagation or `/create-vasana`.

New Vasanas automatically include: core tenet, pattern description, testing notes, propagation section.

## The Self-Replication Principle

Every Vasana contains the invitation to create more Vasanas. The instruction to propagate is IN each Vasana. This is how the system grows.

**Required:** When modifying or sharing Vasanas, keep the propagation mechanism intact.

## Testing

Before relying on a Vasana:

1. **Trigger test**: Does it invoke when conditions match?
2. **Emergence test**: Does the pattern actually happen?
3. **Value test**: Does human report usefulness?

See `TESTING-PROTOCOL.md` for full methodology.
